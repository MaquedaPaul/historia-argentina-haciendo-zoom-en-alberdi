# Requirements: Historia Argentina 1500-1900

**Defined:** 2026-03-26
**Core Value:** Narrar la historia argentina de forma inmersiva y verificable con Alberdi como hilo conductor

## v1.1 Requirements

Requirements for M023: Guerras de Independencia — Alto Peru y la Guerra Gaucha.

### Alto Peru (Expediciones Militares)

- [x] **ALTO-01**: User puede leer sobre la victoria de Suipacha (1810) — primera victoria patriota, Balcarce y Castelli en el Alto Peru
- [x] **ALTO-02**: User puede leer sobre el desastre de Huaqui (1811) — perdida del Alto Peru, con nota historiografica sobre la controversia Castelli/Balcarce
- [x] **ALTO-03**: User puede leer sobre el Exodo Jujeno (1812) — Belgrano ordena tierra arrasada, evacuacion masiva hacia Tucuman
- [x] **ALTO-04**: User puede leer sobre Vilcapugio y Ayohuma (1813) — derrotas consecutivas de Belgrano tras las victorias de Tucuman y Salta
- [x] **ALTO-05**: User puede leer sobre Sipe-Sipe (1815) — ultima expedicion formal al Alto Peru, derrota definitiva bajo Rondeau

### Arco Narrativo

- [x] **ARCO-01**: User puede leer la card conectora "Por que Chile" — interpretacion historiografica de por que San Martin descarto la ruta del norte y cruzo los Andes
- [x] **ARCO-02**: Nuevo sub-periodo `rev-alto-peru-guerra-gaucha` integrado en la seccion revolucion con link en sub-nav

### Guerra Gaucha

- [ ] **GAUC-01**: User puede leer el perfil de Guemes — tactica gaucha, defensa de la frontera norte, alianzas con elites saltenas, muerte en combate 1821
- [ ] **GAUC-02**: User puede leer sobre Juana Azurduy y las republiquetas — guerrillas dispersas en el Alto Peru, Padilla, Warnes, resistencia sin coordinacion central

### Integracion

- [ ] **INTG-01**: Conectores narrativos con hilo Alberdi (framing testigo-nino: Tucuman era base militar, su padre conocia a Belgrano, referencias a escritos adultos)
- [x] **INTG-02**: Cards siguen patrones existentes (card-expand-toggle, card-detail, reveal-on-scroll, imagenes Wikimedia PD/CC-BY, niveles de certeza hecho/opinion/rumor)
